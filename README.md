A framework for building streaming applications.

___

## Quick Start

Install with npm:

```bash
npm install --save brookjs
```

By default, the `package.json` points to the CommonJS version (`cjs/`) for node compatibility, but an ES6 module version (`es/`) and a UMD version (`dist/`) are also provided.
